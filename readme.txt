first of all, install all the required libraries and dependencies by pip install -r requirements.txt

To run the exploratory data analysis python notebook, connect to the kernel and just run directly depends on your needs, or else the results
can be seen in the eda.ipynb

To run the first recommender system, just run python rs1.py, no computation or training is needed as the model is pre-trained and stored using pickle

To run the second recommender system, just run python rs2.py, again no computation or training is needed as the model is pre-trained and stored using pickle

To see evaluation metric results, just run run_eval.py
